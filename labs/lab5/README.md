# lab 5

1. R0=0x8D2, R1=0x68D

- R0 = 1000 1101 0010
- R1 = 0110 1000 1101
- `AND R2, R0, R1`: R2 = R0 & R1
  - R2 = 0000 1000 0000
  - R2 = 0x080
- `EOR R3, R0, R1`: R3 = R0 ^ R1
  - R3 = 1110 0101 1111
  - R3 = 0xE5F
- `ORR R4, R0, R1`: R4 = R0 | R1
  - R4 = 1110 1101 1111
  - R4 = 0xEDF

2. LSL and LSR

- `MOV R0, #14`
  - r0 = 14
- `MOV R1, R0, LSL #3`
  - r1 = r0 << 3
  - r1 = 14 << 3
  - r1 = 1110 << 3
  - r1 = 1110000
  - r1 = 16 + 32 + 64
  - r1 = 112
- `MOV R2, R0, LSR #2`
  - r2 = r0 >> 2
  - r2 = 14 >> 2
  - r2 = 1110 >> 2
  - r2 = 0011
  - r2 = 3
