## part 2

- R0 = 0xC2 = 12 2 = 1100 0010
- R1 = 0x8A = 8 10 = 1000 1010
- ADD R2, R0, R1
  - R2 = R0 + R1 = 101001100 = 0x14C
- SUB R3, R0, R1
  - R3 = R0 - R1 = 11000010 + 01110101 + 1
  - R3 = 100110111 + 1
  - R3 = 00111000 = 0x38
- AND R4, R0, R1
  - R4 = R0 & R1 = 11000010 & 10001010
  - R4 = 10000010 = 0x82
- EOR R5, R0, R1
  - R5 = R0 ^ R1 = 11000010 ^ 10001010
  - R5 = 01001000 = 0x48
